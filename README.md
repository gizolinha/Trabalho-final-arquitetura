# Comparação entre desempenho RISCV e MIPS-32
Este repositório contém todos os códigos e arquivos de teste utilizados para a estruturação do trabalho de comparação entre **duas arquiteturas (MIPS e RISCV)** em relação a implementação de estruturas de dados.
Usando como base os materiais sobre RISCV disponibilizados pelo professor e que podem ser visualizados [aqui](http://riscvbook.com/portuguese/)
